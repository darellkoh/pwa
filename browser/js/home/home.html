Take a look at some of my favorite projects:



GrabGab

GITHUB ·LIVE ·VIDEO DEMO

Co-Developer | Group Project

A nautical-themed, interactive word game inspired by Boggle that supports live, real-time multiplayer activity. Utilized JavaScript, Sequelize, Express, AngularJS, Node.js, Socket.io, HTML5, SCSS and Mocha.



MeetupTrackr

GITHUB ·LIVE

Sole Developer | Personal Side Project

As an avid user and a big fan of Meetup.com, I built a web application that allows users to track real-time live feeds of other cool meetups that other people are attending worldwide, searchable by topics of interest and cities. Utilized JavaScript, React, Appbase.io's API, Meetup.com's RSVP API.



WeatherMonitor

GITHUB ·LIVE

Sole Developer | Personal Side Project

A weather monitoring app that returns data visualizations of current temperatures and a 7-day forecast at 3-hour intervals based on user location input. Built using React, Redux, OpenWeatherMap API and Jest (for testing).



Space Blitz

GITHUB ·LIVE

Sole Developer | Hackathon Project

A 2D vertical scroller space shooter game created during a weekend-long hackathon, inspired by the childhood arcade game favorite, Galaga. Utilized JavaScript, Express, Node.js, HTML5 and Phaser.



World Wide Whiteboard

GITHUB ·LIVE

A live drawing board application built using event emitters and Socket.io that enables users to draw together with friends in real-time. Utilized JavaScript, HTML5, CSS, Bootstrap, Node.js, Express and Socket.io.



Game of Life

GITHUB ·LIVE

A JavaScript implementation of the popular Conway's Game of Life, an example of cellular automaton played on a 2D rectangular grid of cells. Utilized JavaScript, HTML5, CSS and Bootstrap.
